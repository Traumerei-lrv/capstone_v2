Problem 1 - Reactor Temperature Scan

Sample story:

Alert: Reactor overheating! The ship’s system stored temperature readings across different reactor sections. You must scan a segment of the data to find the highest temperature before meltdown.

Sample task:

Find the highest temperature between section p and q.

(This is a recursive problem: Given an array A indexed from p to q containing integers, return the integer in A [p..q] with the maximum value.)
