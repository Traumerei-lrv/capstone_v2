DSA Mission Brief

The maze is stored as a 2D array. Each coordinate is treated as a node. The ship can only move to neighboring nodes: up, down, left, or right. Safe tiles, blocked tiles, danger tiles, and the exit are all stored as values inside the array. The goal is to find a valid route from the start node to the exit node.
